## 1. 常用命令
- 安装gitbook：npm install gitbook-cli -g
- 创建文件夹：mkdir mybook
- 进入文件夹：cd mybook
- 初始化：gitbook init
- 编译：gitbook build
- 启动服务：gitbook serve
- 预览：http://localhost:4000/
- 查看gitbook版本：gitbook -V（V大写）
